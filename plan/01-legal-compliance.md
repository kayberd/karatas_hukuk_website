# 01 — Legal & Compliance

Governs all other documents. Read first.

## Sources

- Avukatlık Kanunu m.55 (reklam yasağı ana çerçevesi)
- TBB Reklam Yasağı Yönetmeliği (last major change 09.08.2024)
- TBB Meslek Kuralları
- KVKK (6698) — kişisel verilerin korunması

## Allowed site content (closed list — nothing outside it)

- Name, surname, academic title (if any)
- Photo
- Other lawyers in office: name + academic title
- Avukatlık ortaklığı → registered (tescilli) name
- Avukatlık bürosu → büro name
- TBB + baro registration (sicil) numbers
- Profession start date
- University graduated from
- Known foreign languages
- Office address, phone, fax
- E-posta + KEP (kayıtlı elektronik posta)

> Allowed extras: anonymized legal articles/scientific work; activity areas described as
> **faaliyet alanı** (NOT "uzmanlık"); password-protected backend "sanal ofis".

## Forbidden

- "Referanslar" / client names / case results / testimonials
- Banned phrases: "en iyi", "garanti", "kesin sonuç", "uzman" (as expertise claim), discount / campaign / "ücretsiz danışmanlık"
- Paid Google ads, sponsored social posts, paid ranking, click-buying
- Redirect shortcuts to/from the site for lead-gen
- Keywords beyond: own name, firm/partnership name, city + registered baro, and only:
  `avukat, hukuk, hukukçu, adalet, savunma, iddia, eşitlik, hak`

## Domain rule

- Site domain MUST use **`av.tr`** extension. No `.com` / `.com.tr` as primary.
- Apply via nic.tr (av.tr requires baro/lawyer verification).

## Enforcement note

Since 09.08.2024 there is **no warning step** — violations trigger direct disciplinary
investigation. Treat compliance as a hard gate, not a nice-to-have.

## KVKK requirements

- Aydınlatma metni (clear, linked from forms + footer)
- Explicit consent checkbox on contact form (unchecked by default)
- Cookie consent banner + cookie policy
- Document where contact-form data is stored (prefer TR/EU)
- Sır saklama: no client data exposed anywhere public

## Compliance checklist (pre-launch gate)

- [ ] Domain is `av.tr`
- [ ] No banned phrases anywhere (incl. meta tags, alt text)
- [ ] No testimonials / case results / client names
- [ ] Activity areas worded as "faaliyet alanı"
- [ ] No paid ads / sponsored links configured
- [ ] KVKK aydınlatma metni published + linked
- [ ] Contact form consent checkbox present
- [ ] Cookie banner active
- [ ] Lawyer profiles use only allowed fields
