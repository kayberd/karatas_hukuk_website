# 04 — Design & UX

## Visual identity

- Palette: koyu lacivert + bordo + altın/antrasit accents (classic premium)
- Typography: serif (geleneksel) or modern sans-serif (çağdaş) — pick one, consistent
- Generous white space (minimal, premium perception)
- Restrained legal iconography (terazi, kanun kitabı) — no clip-art
- Logo: firm mark, used in header + footer

## Layout system

- Header: logo + nav + required identity (büro adı)
- Footer: address, phone, e-posta, KEP, sicil no, KVKK + gizlilik + çerez links
- Reusable components: lawyer card, article card, activity-area card, CTA-info block (non-ad)
- Section spacing rhythm consistent across pages

## UX principles

- Information-first, low friction
- Clear path: who → what areas → articles → contact
- No aggressive popups / lead-capture modals
- Contact form short, with visible KVKK consent

## Accessibility (a11y)

- Semantic HTML, heading order
- Color contrast AA minimum
- Keyboard navigable
- Alt text on images (no banned phrases in alt)
- Focus states visible

## Responsive

- Mobile-first
- Breakpoints: mobile / tablet / desktop
- Tap targets >= 44px
- Test nav, forms, map on mobile

## Performance targets (tie to 07)

- Core Web Vitals green (LCP, CLS, INP)
- Optimized images (next-gen formats)
- Minimal JS for a brochure site
