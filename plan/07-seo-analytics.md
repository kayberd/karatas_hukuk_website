# 07 — SEO & Analytics (ban-safe)

All SEO must respect TBB ban (see `01`). Technical SEO is the safe zone;
manipulative keyword stuffing and paid ranking are forbidden.

## Allowed technical SEO

- Clean titles + meta descriptions (no banned phrases)
- XML sitemap + robots.txt
- Semantic HTML, proper heading structure
- Fast load / Core Web Vitals
- Mobile-friendly
- Canonical URLs
- Image alt text (compliant wording)

## Schema markup

- `Attorney`
- `LegalService`
- `FAQPage` (for FAQ sections)
- `Article` (for bilgi bankası)

## Keyword limits (hard rule)

Only: own name, firm/partnership name, city + registered baro, and:
`avukat, hukuk, hukukçu, adalet, savunma, iddia, eşitlik, hak`

No other promotional keywords. No paid ads. No sponsored placement.

## Google Business Profile

- Allowed and high-value for local visibility
- Keep info accurate + compliant (no banned phrases)

## Analytics

| Option | Notes |
| --- | --- |
| Plausible / Umami (recommended) | Cookieless, KVKK-friendly, lighter consent burden |
| GA4 | Heavier consent requirements |

Recommendation: **cookieless analytics** to simplify KVKK compliance.

## Forbidden (repeat)

- Paid Google/social ads
- Click-buying, paid ranking
- Hidden keywords / redirect shortcuts
- Banned phrases in any meta/SEO field
